Generated 
=========













### From Xcri 







Jump to: [navigation](Generated.html#column-one),
[search](Generated.html#searchInput)



*URI*: 

*Definition*: The optional generated attribute provides a mechanism for
indicating when the XCRI feed was last generated by the provider, and is
used as part of the [delta update
pattern](Delta_update_pattern.html "Delta update pattern").

*Type*: A date or time formatted in accordance with ISO 8601, as used in
the definition of the W3C XML Schema "DateTime" type OR "Date" type.

*Namespace*: This is defined within the namespace:



\[[edit](../index.php@title=Generated&action=edit&section=1.html "Edit section: Example")\] Example
---------------------------------------------------------------------------------------------------------------------------------------------------------------------

A datetime formatted in accordance with [ISO
8601](http://www.iso.org/iso/support/faqs/faqs_widely_used_standards/widely_used_standards_other/date_and_time_format.htm "http://www.iso.org/iso/support/faqs/faqs_widely_used_standards/widely_used_standards_other/date_and_time_format.htm"){.external
.text}, as used in the definition of the W3C XML Schema "DateTime" type.

    generated="2010-05-19T13:39:59"


\[[edit](../index.php@title=Generated&action=edit&section=2.html "Edit section: Guidance Notes")\] Guidance Notes
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

This attribute is only used with the [delta update
pattern](Delta_update_pattern.html "Delta update pattern"). The [simple
aggregation
pattern](Simple_aggregation_pattern.html "Simple aggregation pattern")
does not use this attribute.
